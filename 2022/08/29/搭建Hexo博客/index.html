搭建Hexo博客

搭建hexo博客

环境搭建

1.安装Node.js

打开nodejs官网https://nodejs.org/en/，根据系统版本选择下载并安装Node,js,安装完成后，打开cmd，输入`node -v`，如果能正确显示Node.js版本，则说明安装成功

2.通过npm安装Hexo框架

打开cmd，输入npm install -g hexo-cli,安装完成后输入hexo -v查看hexo版本。（淘宝的npm代码仓库：npm install -g cnpm --registry=https://registry.npm.taobao.org）

3.安装gitforwin

https://gitforwindows.org/，安装完成后打开cmd，输入`git`查看是否成功安装![](https://yuki-picgo-img.oss-cn-beijing.aliyuncs.com/PicGo/20220829195109.png)

本地使用

1.创建博客目录

创建一个目录用于保存博客文件

2.初始化博客

打开Git CMD，进入博客目录，输入hexo init初始化博客

3.启动博客

hexo server

启动成功

4.操作博客

常用操作：创建博客、启动服务、Generate 、部署（可只写首字母）

博客部署

1.生产github.io仓库

登录GitHub，创建一个新的public仓库，仓库名必须为YourGitHubName.github.io

2.在博客目录下装git部署插件

在目录中输入npm install --save hexo-deployer-git

3.修改本地_config.yml文件

4.部署到远端

设置用户名和邮箱

设置完成之后hexo -d部署，使用仓库名即可访问
